# 🎮 LIXCHINI

Jogo arcade 2D desenvolvido em Python com a biblioteca **Pygame**, como projeto acadêmico durante o curso de Análise e Desenvolvimento de Sistemas. O objetivo foi aplicar na prática conceitos de lógica de programação, estruturação de código e persistência de dados em um contexto diferente do meu foco principal (Machine Learning), como exercício de aprendizado.

## 📋 Sobre o projeto

- Jogo com **2 níveis**, cada um encerrado por um evento de tempo limite (timeout)
- Suporte a **modo cooperativo** (2 jogadores simultâneos)
- Pontuação salva em **banco de dados SQLite3**, implementada utilizando o **padrão de projeto Proxy**

## 🛠️ Tecnologias utilizadas

- Python
- Pygame
- SQLite3

## 🚀 Como executar

1. Clone este repositório:
   ```bash
   git clone https://github.com/amandaliisboa/lixchini.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd lixchini
   ```
3. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
4. Execute o jogo:
   ```bash
   python main.py
   ```

## 🧠 Conceitos aplicados

- Lógica de jogos com Pygame (loop principal, eventos, colisões)
- Estruturação de código em múltiplos arquivos/pastas (organização em `code` e `asset`)
- Persistência de dados com SQLite3
- Padrão de projeto **Proxy** para acesso ao banco de dados
- Lógica para múltiplos jogadores (modo cooperativo)
